# План модульного тестирования: arccos(x) (разложение в степенной ряд)

## 1. Контракт
- Область определения: x ∈ [-1; 1]
- Область значений: y ∈ [0; π]
- Некорректные входные данные (|x| > 1, NaN, ±Inf) → IllegalArgumentException [ASSUMPTION]

## 2. Допущения по реализации (для численной устойчивости) [ASSUMPTION]
- Вычисление через ряд для arcsin(x):
  - arccos(x) = π/2 − arcsin(x)
- Для x, близких к границам, используются устойчивые преобразования:
  - при x ≈ 1: arccos(x) = 2 * arcsin( sqrt((1 − x)/2) )
  - при x ≈ −1: arccos(x) = π − 2 * arcsin( sqrt((1 + x)/2) )

## 3. Параметры вычисления ряда
- eps = 1e−10
- maxTerms = 10000
- Критерий остановки: |term| < eps

## 4. Тестовое покрытие
### 4.1. Классы эквивалентности
- A: x ∈ (−1; 1), типичные значения
- B: x около 0
- C: x близко к 1
- D: x близко к −1
- E: граничные значения x = −1 и x = 1
- F: некорректные значения (вне области определения / NaN / Inf)

### 4.2. Набор тестовых точек
- Типичные: −0.9, −0.5, −0.1, 0, 0.1, 0.5, 0.9
- Границы: −1, 1
- Около границ: 1−1e−12, 1−1e−8, −1+1e−12, −1+1e−8
- Некорректные: 1+1e−6, −1−1e−6, NaN, +Inf, −Inf

## 5. Оракул и допуски
- Оракул: сравнение с Math.acos(x)
- Допуски:
  - внутри области (не у границ): |err| ≤ 1e−9
  - около границ: |err| ≤ 1e−7

## 6. Дополнительные проверяемые свойства
- Монотонность: при x1 < x2 выполняется arccos(x1) > arccos(x2)
- Симметрия: arccos(−x) = π − arccos(x)
- Диапазон: результат ∈ [0; π]
